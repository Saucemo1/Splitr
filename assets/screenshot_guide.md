# App Store Screenshots Guide

## Required Screenshot Sizes:

### iPhone Screenshots:
- **iPhone 15 Pro Max (6.7"):** 1290 x 2796 pixels
- **iPhone 15 Pro (6.1"):** 1179 x 2556 pixels  
- **iPhone 15 (6.1"):** 1179 x 2556 pixels
- **iPhone 15 Plus (6.7"):** 1290 x 2796 pixels

## How to Take Screenshots:

1. **Run the app on iPhone Simulator:**
   ```bash
   flutter run -d [simulator-id]
   ```

2. **Take screenshots of these key screens:**
   - **Screen 1:** Main screen with "Add Bill" section
   - **Screen 2:** Upload panel with camera/upload options
   - **Screen 3:** Bill scanning/processing screen
   - **Screen 4:** Add people section with colorful chips
   - **Screen 5:** Item assignment with split options
   - **Screen 6:** Final split results with totals

3. **Use iPhone Simulator:**
   - Device → Screenshot (Cmd+S)
   - Or use the screenshot button in simulator

## Screenshot Tips:
- Show the app in portrait mode
- Make sure all text is readable
- Show the beautiful UI we created
- Highlight the AI scanning feature
- Show the responsive design

## Alternative: Use Xcode Simulator
1. Open Xcode
2. Window → Devices and Simulators
3. Choose iPhone 15 Pro Max
4. Run your app
5. Take screenshots (Cmd+S)
